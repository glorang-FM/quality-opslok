RENDER ENVIRONMENT VARIABLE — ADD THIS
=======================================

In your Render dashboard → Quality OpsLok server service → Environment:

Add this new variable:

  ANTHROPIC_API_KEY = <your Anthropic API key>
  (get it from: console.anthropic.com → API Keys)

This is required for the AI document parsing feature (Documents → AI Parse).
All other existing env vars (DATABASE_URL, JWT_SECRET, CLIENT_URL, NODE_ENV) stay the same.
